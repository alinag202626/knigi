# Book Tracker

**Автор:** [Ваше Имя Фамилия]

## Описание

Приложение для управления списком прочитанных книг с возможностью фильтрации и сохранения данных в JSON‑формате.

## Функциональность

- Добавление книг с указанием названия, автора, жанра и количества страниц
- Просмотр списка книг в табличном виде
- Фильтрация по жанру и количеству страниц
- Сохранение данных в файл `books.json`
- Загрузка данных при запуске приложения
